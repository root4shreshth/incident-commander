Welcome to IncidentCommanderEnv 
You are an on-call SRE. Select an incident scenario and click "Start Incident".

Your goal: diagnose the root cause and resolve the incident.

Available actions:
  - list_services: See all services
  - describe_service: Deep dive into a service
  - read_logs: Read service logs
  - check_metrics: CPU, memory, latency, errors
  - restart_service: Restart (with optional new config)
  - rollback_deployment: Rollback to a previous version
  - run_diagnostic: check_connectivity, check_health, check_resources
  - update_config: Change runtime config
  - resolve_incident: Declare incident resolved
